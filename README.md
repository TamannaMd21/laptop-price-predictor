# 💻 Laptop Price Predictor

## 📌 Overview

The **Laptop Price Predictor** is a Machine Learning project that estimates the price of a laptop based on its specifications such as brand, RAM, processor, storage, and more.

This project helps users understand how different features affect laptop pricing and provides a quick prediction using a trained model.

---

## 🚀 Features

* Predict laptop price based on user input
* Clean and user-friendly interface
* Data preprocessing and feature engineering
* Machine Learning model for accurate predictions
* Real-time predictions using Streamlit

---

## 🛠️ Tech Stack

* **Python**
* **Pandas & NumPy** (Data Processing)
* **Scikit-learn** (Model Building)
* **Streamlit** (Web App Interface)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/TamannaMd21/laptop-price-predictor.git
cd laptop-price-predictor
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the application

```bash
streamlit run app.py
```

## 📊 How It Works

1. User inputs laptop specifications
